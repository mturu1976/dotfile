# tmux チートシート

## 基本操作
プレフィックスキー: `Ctrl+j`

## ウィンドウ・ペイン操作
| キー | 説明 |
|------|------|
| `prefix + c` | 新しいウィンドウを作成（現在のディレクトリ） |
| `prefix + n` | 新しいセッションを作成 |
| `prefix + |` | 垂直分割（現在のディレクトリ） |
| `prefix + -` | 水平分割（現在のディレクトリ） |
| `prefix + z` | ペインをズーム/解除 |
| `prefix + !` | ペインを新しいウィンドウに移動 |
| `prefix + {/}` | ペインの入れ替え |
| `prefix + </>`| ウィンドウの入れ替え |

## ナビゲーション
| キー | 説明 |
|------|------|
| `Ctrl+h/j/k/l` | Vim/tmux間のシームレスな移動 |
| `prefix + h/j/k/l` | ペイン間の移動 |
| `prefix + H/J/K/L` | ペインサイズの変更（大） |
| `prefix + M-h/j/k/l` | ペインサイズの変更（小） |
| `prefix + s` | tmux-jump（ペインジャンプ） |
| `prefix + o` | sessionx（セッション管理） |

## ポップアップ機能
| キー | 説明 |
|------|------|
| `prefix + p` | フローティングターミナル |
| `prefix + g` | lazygit起動 |
| `prefix + t` | tig起動 |
| `prefix + N` | クイックノート |
| `prefix + T` | 一時的なターミナル |

## 高度な機能
| キー | 説明 |
|------|------|
| `prefix + S` | ペイン同期モード ON/OFF |
| `prefix + P` | ペイン履歴を保存 |
| `prefix + v` | クリップボードからペースト |
| `prefix + R` | 最後のコマンド再実行 |
| `prefix + m` | マウスモード切り替え |
| `prefix + D` | ペインサイズ表示 |
| `prefix + r` | 設定再読み込み |

## コピーモード（Emacsキーバインド）
| キー | 説明 |
|------|------|
| `prefix + [` | コピーモード開始 |
| `M-w` | 選択範囲をコピー |
| `C-w` | 選択範囲をコピー |
| `Enter` | 選択範囲をコピー |
| `Escape` | コピーモードを終了 |

## 特殊機能
| キー | 説明 |
|------|------|
| `Ctrl+f` | tmux-fzf（ファジー検索） |
| `Ctrl+\` | tmux-menus（ビジュアルメニュー） |
| `prefix + Tab` | tmux-sidebar（ファイルツリー） |
| `Space` | tmux-thumbs（URL/ハッシュコピー） |

## レイアウトプリセット
| キー | 説明 |
|------|------|
| `prefix + M-1` | 横均等分割 |
| `prefix + M-2` | 縦均等分割 |
| `prefix + M-3` | メイン横分割 |
| `prefix + M-4` | メイン縦分割 |
| `prefix + M-5` | タイル状 |

## ヘルプ・リファレンス
| キー | 説明 |
|------|------|
| `prefix + ?` | 全キーバインド一覧 |
| `prefix + /` | キーバインド検索 |
| `prefix + F1` | このチートシート表示 |

## ステータスバー表示
- 左側: ⚡ セッション名
- 中央: ウィンドウ一覧（🔍はズーム中）
- 右側: Gitブランチ | CPU使用率 | バッテリー | 日時

## プラグイン管理
| キー | 説明 |
|------|------|
| `prefix + I` | プラグインインストール |
| `prefix + U` | プラグインアップデート |
| `prefix + M-u` | プラグインアンインストール |
