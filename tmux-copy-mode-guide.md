# tmuxコピーモード（Emacsモード）ガイド

## コピーモードの開始
- `Ctrl+j [` - コピーモードを開始

## コピーモード中の操作

### カーソル移動
- `↑/↓/←/→` - カーソル移動
- `Ctrl+p/n` - 上下移動（Previous/Next）
- `Ctrl+b/f` - 左右移動（Back/Forward）
- `Alt+b/f` - 単語単位で移動
- `Ctrl+a/e` - 行頭/行末へ移動
- `Alt+</>`  - バッファの最初/最後へ
- `Ctrl+u/d` - 半ページ上下スクロール
- `PageUp/PageDown` - 1ページ上下スクロール

### 選択とコピー
- `Ctrl+Space` - 選択開始
- `Alt+w` - 選択範囲をコピーしてコピーモード終了
- `Ctrl+w` - 選択範囲をコピーしてコピーモード終了
- `Enter` - 選択範囲をコピーしてコピーモード終了
- `Escape` - コピーモードを終了（コピーせず）

### 検索
- `Ctrl+s` - 前方検索
- `Ctrl+r` - 後方検索
- `n` - 次の検索結果
- `N` - 前の検索結果

## ペースト
- `Ctrl+j ]` - コピーしたテキストをペースト

## マウス操作
- マウスドラッグ - テキスト選択（自動的にコピー）
- マウスホイール - スクロール

## Extraktoプラグイン
- `Ctrl+j Tab` - 画面内のテキスト（URL、パス、単語など）を抽出してfzfで選択

## 備考
現在の設定ではEmacsキーバインドを使用しているため、Emacsユーザーには馴染みやすい操作になっています。
