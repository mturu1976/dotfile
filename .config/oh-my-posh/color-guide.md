## カスタマイズ方法

1. テーマファイルを編集
```bash
nvim ~/dotfile/.config/oh-my-posh/themes/gruvbox-muted.omp.json
```

2. 色を変更
- `"background": "#色コード"`
- `"foreground": "#色コード"`

3. 反映
```bash
exec zsh
```

## 彩度調整のコツ

### 彩度を下げる
- 元の色のRGB値を均等に近づける
- グレーを混ぜる感覚で

### 例：
- 鮮やか: `#FF0000` (赤)
- 彩度50%: `#BF4040`
- 彩度25%: `#9F6060`

### HSL変換ツール
https://www.w3schools.com/colors/colors_hsl.asp
でSaturation（彩度）を調整
