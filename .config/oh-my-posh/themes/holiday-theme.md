# Holiday Theme 🎄

A warm and cozy color theme inspired by holiday evenings.

## Color Palette

### Primary Colors
- `#26111F` - **Deep Wine** (darkest purple, background)
- `#2A1440` - **Midnight Purple** (active pane background)
- `#593B40` - **Cocoa Brown** (borders, inactive elements)
- `#A67165` - **Cinnamon** (accents, active elements)
- `#BFA393` - **Warm Sand** (text, foreground)

## Usage

### Oh My Posh
```bash
eval "$(oh-my-posh init zsh --config ~/dotfile/.config/oh-my-posh/themes/holiday.omp.json)"
```

### tmux
The tmux configuration is automatically styled to match the Holiday theme colors.

## Inspiration
This theme evokes the feeling of cozy holiday evenings with its warm browns and deep purples, 
like sitting by a fireplace with hot cocoa.

## Customization
To adjust the colors, edit:
- Oh My Posh: `~/dotfile/.config/oh-my-posh/themes/holiday.omp.json`
- tmux: `~/.tmux.conf` (search for "カスタムパレット")
